# 投影模式 · 测试清单（人工验证）

> 覆盖当前投影模式全部功能。**打勾项**建议在真机上逐条走一遍，尤其标 🖥️ 的项目在自动化里测不了、必须人工验。
> 建议至少覆盖：桌面 Chrome / Edge（主力）、macOS Safari、iPad Safari、Android Chrome。

## A. 入口

- [ ] 主页 hero「进入投影模式」按钮 → 打开演示台
- [ ] 侧栏「投影模式」→ 打开演示台（与 hero 同一个）
- [ ] 详情页某首歌「投影」按钮 → 打开单曲控制面板（该曲已在其中）
- [ ] 图片曲谱歌曲（无结构化简谱）详情页「投影」按钮应置灰，点/hover 提示「该曲目暂不支持投影」

## B. 演示台（控制端，屏1）

- [ ] 诗歌库搜索：输入歌名/歌手能过滤
- [ ] 点「＋」把歌加入本堂歌单
- [ ] 本堂歌单：↑ ↓ 排序、✕ 删除都正常
- [ ] 缩略图列表：每页一张缩略图，点缩略图跳到该页（当前页高亮）
- [ ] 上一页 / 下一页按钮 + 页码「第 X 页 / 共 Y 页」正确
- [ ] 键盘 ← → / 空格翻页；Esc 关闭演示台；B 黑屏
- [ ] 关闭演示台再打开，**本堂歌单还在**（持久化）

## C. 显示内容 / 排版

- [ ] 四个开关（和弦 / 简谱 / 歌词 / 拼音）任意组合，关掉的层不占位、画面重排
- [ ] 拼音：每段歌词各自上方一行拼音；生僻字/多音字标注可能不准（可接受，不应报错或乱码）
- [ ] 段落标记（Chorus / Bridge / Pre-Chorus / 主歌 / 副歌等）**不显示**在投影
- [ ] 每首歌**第一页顶部**显示「{调} {拍号} + 歌名」，后续页不重复；跟随移调变调
- [ ] 详情页移调后，若正投影这首，投影内容跟着换调
- [ ] 字号滑块：往大拉每页放更少行、字更大；实时同步；刷新后设置还在
- [ ] 比例 16:9 / 4:3 切换后重新分页
- [ ] **加经文后诗歌分页不受影响**（诗歌不会变成一行一页）✅ 已修，仍建议复验

## D. 经文（Bible）

- [ ] 经文框输入「诗篇 23:1-6」→「加经文」→ 联网取回并加入歌单（📖 标记）
- [ ] 支持中文经卷名 / USFM 缩写（GEN）/ 拼音（csj）；格式「书 章:节-节」
- [ ] 经文投影：标题 + 金色节号 + 白字大字，可读
- [ ] 🖥️ 断网时加经文 → 提示「经文服务暂时不可用」，不崩
- [ ] 错误引用（如「abc 99:99」）→ 友好报错

## E. 背景 / 外观

- [ ] 背景：暖阳 / 深蓝 / 纯黑 一键切换，投影与预览同步
- [ ] 🖥️ 上传图片：选一张本地图 → 投影铺满显示（cover），控制端预览同步
- [ ] 🖥️ 上传后刷新页面，背景图仍在（存 IndexedDB）
- [ ] 白字 + 阴影在暖色/复杂背景图上都清晰可读

## F. 双屏 / 投影窗口（🖥️ 必须真机 + 外接屏）

- [ ] 🖥️ 单屏时点「进入投影」→ 新窗口打开，可手动全屏
- [ ] 🖥️ 接了外接显示器/投影仪 → 点「进入投影」→ 浏览器请求「窗口管理」权限 → 允许后投影窗口**自动定位到副屏并全屏**
- [ ] 🖥️ 若自动全屏没生效，点控制端「投影全屏」补触发
- [ ] 🖥️ 黑屏按钮 → 投影窗口变全黑；再点恢复
- [ ] 🖥️ 关闭投影窗口 → 控制端状态恢复「未开启」
- [ ] 🖥️ 关演示台/详情页 → 投影窗口**不被强制关**（避免误触中断）
- [ ] 🖥️ Safari / 不支持窗口管理的浏览器 → 降级为普通新窗口 + 提示，仍可手动拖到副屏并全屏

## G. 息屏 / 长时间投影（🖥️）

- [ ] 🖥️ 投影窗口全屏后，静置超过系统息屏时间，屏幕**不息屏**（Wake Lock 生效；不支持则需手动调长系统息屏）
- [ ] 🖥️ 切到别的窗口再切回投影，Wake Lock 自动重申请

## H. 边界情况

- [ ] 歌单为空 / 全部删除时，投影页显示「歌单为空…」提示，不留旧画面
- [ ] 一首歌只有 1-2 行 → 投影不出现突兀大片留白（内容居中）
- [ ] 超长行（一行很多小节）→ 自动缩小铺满、不横向溢出（字会偏小，可用字号滑块或拆分应对）
- [ ] 手机当控制端（窄屏 <820px）→ 三栏自动堆叠，所有控件可用
- [ ] 歌单里引用了已删除的歌 → 该项跳过、不崩

## I. 离线（PWA）

- [ ] 装成 PWA / 首次联网访问后，断网仍能打开 App 与 projection.html（经文与背景图联网内容除外）
