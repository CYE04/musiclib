# 投影模式 · 使用说明（给同工）

诗歌库的「投影模式」可以把诗歌简谱/歌词、经文，全屏投到投影仪或电视上，供会众敬拜时看。
下面是最常用的操作，照着做就行，不需要懂技术。

---

## 一、准备：接好第二块屏

1. 电脑接上投影仪 / 电视（HDMI 或无线投屏）。
2. 在电脑「显示设置」里选 **扩展模式**（不是「镜像」）——这样电脑屏是你操作的，投影屏是给会众看的。
   - 只有一块屏也能用，只是投影会盖住你的操作画面。

## 二、打开演示台

- 在诗歌库**主页**，点 **「进入投影模式」**（或左侧菜单的「投影模式」）。
- 会打开一个「演示台」，分三块：**诗歌库**、**本堂歌单**、**演示控制**。

## 三、排好这场要用的诗歌 / 经文

1. 左边**诗歌库**搜索歌名，点歌右边的 **＋** 加进「本堂歌单」。
2. 想加**经文**：在诗歌库下方的框里输入，例如 `诗篇 23:1-6`，点 **「加经文」**。
   - 经卷名可以打中文（诗篇）、也可以打拼音首字母。
3. 「本堂歌单」里可以用 **↑ ↓** 调顺序、**✕** 删除。
   - 歌单会自动记住，下次打开还在。

## 四、开始投影

1. 点右上角 **「进入投影」**。
2. 第一次会弹出**权限请求**（允许管理窗口位置），点**允许**。
3. 投影画面会自动跑到投影那块屏并全屏。
   - 如果没自动全屏：点控制端的 **「投影全屏」**，或在投影画面上按一下也行。

## 五、演示时的操作（都在你这块屏上）

- **翻页**：点「上一页 / 下一页」，或按键盘 **← →**、空格。也可以直接点某一页的**小缩略图**跳过去。
- **黑屏**：讲道、祷告时想清空投影，点 **「黑屏」**；再点恢复（键盘按 **B** 也行）。
- **显示内容**：上面四个开关——**和弦 / 简谱 / 歌词 / 拼音**，想显示哪层就开哪层。
- **字号**：底部「字号」滑块，往右字更大（每页放的行会变少）。
- **背景**：可选 暖阳 / 深蓝 / 纯黑，或 **「上传图片」**用自己的背景图。
- **比例**：投影仪是宽屏选 **16:9**，老式方屏选 **4:3**。

## 六、结束

- 关掉投影窗口即可（或按 **Esc** 退出全屏再关）。
- 关演示台不会自动关投影窗口，避免误触中断——需要时手动关投影窗口。

---

## 小贴士 / 常见问题

- **投影没自动到第二屏？** 浏览器可能没拿到「窗口管理」权限，或用的是 Safari。手动把弹出的投影窗口拖到投影那块屏，再点「投影全屏」即可。用 **Chrome / Edge** 体验最好。
- **投影中途息屏了？** 系统息屏时间太短。投影模式会尽量阻止息屏，但少数设备不支持——可先到系统设置把「屏幕关闭时间」调长。
- **背景图太亮、字看不清？** 换深一点的背景，或用纯色背景；文字本身已带描边阴影。
- **经文加不进来？** 检查是否联网，以及引用格式（例：`约翰福音 3:16`）。
- **段落名（副歌/Bridge 等）不显示是正常的**——投影画面刻意保持纯净，只显示谱和词。
